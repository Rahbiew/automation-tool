# AutomateFlow: Automation Savings Blueprint

A self-service automation blueprint and ROI calculator for service professionals. Instantly discover how many hours and dollars you can save by automating repetitive admin tasks, and generate a tailored automation plan.

## Features

- Occupation-based automation savings estimation (hours & $ per year)
- Live software selection and automation plan generation
- Clear business value pitch and milestone project schedule
- Value-based project quote with PPP-adjusted pricing

## Usage

1. Select your occupation, weekly jobs, hourly admin cost, and currency.
2. Choose the software tools you already use.
3. Click "Generate Blueprint" to view time, money savings, and your recommended project plan.
4. Share or use the milestone quote for your team or clients!

## Deployment

This app is deployed on GitHub Pages.  
For changes, push updates to `index.html` on the `main` branch.  
GitHub Pages will rebuild automatically.
